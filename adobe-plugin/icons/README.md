# Plugin Icons

Place plugin icons here with the following sizes:

- `icon_light.png` - 23x23 and 46x46 (@2x) for light theme
- `icon_dark.png` - 23x23 and 46x46 (@2x) for dark theme
- `icon_disabled.png` - 23x23 and 46x46 (@2x) for disabled state

Icons should be PNG format with transparent background.

For now, placeholder icons will be generated automatically by the build script.
